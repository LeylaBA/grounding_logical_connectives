# grounding_logical_connectives
A neural dynamic cognitive architecture that handles sentences with logical connectives AND, OR and XOR. 

"GroundingOR_Thesis.json" file can be opened in CEDAR to view and interact with the architecture. CEDAR can be set up under: https://cedar.ini.rub.de/

"RTs_conference.ipynb" contains the preprocessing and visualization of reaction time data in "exp_file_paper_130".
